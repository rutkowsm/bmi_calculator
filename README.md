Aplikacja do obliczania indeksu BMI.

Dane wejściowe: 
  - masa ciała w kg;
  - wzrost w cm.

Dane wyjściowe: 
  - indeks BMI.

Przyciski: 
  - Caclculate BMI - przelicza indeks BMI i powoduje wyświetlenie wartości w polu TextView;
  - Clear all - czyści pole TextView a także wartości wprowadzone w polach input.
